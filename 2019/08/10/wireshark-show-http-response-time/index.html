wireshark如何显示http响应时间
发布于：2019-08-10 16:05:00 标签：/ wireshark / http / 访问：

显示http响应时间

有时候，我们要检查http响应时间是快是慢。
http请求报文有时间戳，http响应报文也有时间戳，二者相减即可得到，我知道你是这么想的；可是在wireshark里要怎么做呢？
其实wireshark已经帮我们实现了这一点。
第一步：filter中输入http，显示如下图：

第二步：filter中输入http.time，显示如下图：

对比上图，多了[Time since request: 0.1102 seconds]
第三步：选中这个字段，显示右键菜单，如下图：

选择 Apply as Columns，显示如下图：

第四步：filter中输入http.time>5.0，可以只显示响应时间大于5秒的http响应报文

可视化统计规律

http响应时间我们看到了，能否统计显示，通过可视化揭示出一些规律呢？
答案也是可以的。

图形是怎么出来的呢？菜单Statistics，进入IO Graphs。
点击+号按钮，其中的每一列都是可以编辑的，细节不再赘述。
